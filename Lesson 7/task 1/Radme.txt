
Student Management System
Опис
Цей проект реалізує систему управління групою студентів за допомогою JavaScript. 
Вона включає функціональність для додавання, видалення, перегляду та пошуку студентів. 
Проект використовує HTML для інтерфейсу, CSS для стилізації та JavaScript для логіки програми.

Файли проекту
script.js: Основний файл JavaScript, що містить функціональність для управління студентами.
doctype.html: HTML-файл для інтерфейсу користувача.
style.css: CSS-файл для стилізації інтерфейсу.
studentGroup.test.js: Файл тестів для перевірки функціональності системи за допомогою Jest.
Функціональність
Конструктор Student
Student(id, firstName, lastName, age, averageGrade): 
Конструктор для створення об'єкта студента.
Об'єкт StudentGroup
addStudent(id, firstName, lastName, age, averageGrade):
 Додає нового студента до групи. Перевіряє наявність студента з таким же ID.
removeStudent(id): Видаляє студента з групи за ID.
displayStudents(): Відображає всіх студентів у групі.
findStudentById(id): Шукає студента за ID та відображає його інформацію.
displayMessage(message): Відображає повідомлення на екрані.
Функції взаємодії з HTML-формою
addStudent(): Викликає StudentGroup.addStudent() для додавання студента.
removeStudent(): Викликає StudentGroup.removeStudent() для видалення студента.
displayStudents(): Викликає StudentGroup.displayStudents() для відображення всіх студентів.
findStudent(): Викликає StudentGroup.findStudentById() для пошуку студента за ID.
Тестування
Для тестування функціональності використовується Jest. 
Щоб запустити тести, виконайте наступні команди в терміналі:

Встановіть Jest, якщо ще не встановлено:

bash
Копіювати код
npm install --save-dev jest
Запустіть тести:

bash
Копіювати код
npm test
Вимоги
Node.js
npm (Node Package Manager)
Налаштування
Клонуйте репозиторій або завантажте файли проекту.
Переконайтесь, що всі залежності встановлені:
bash
Копіювати код
npm install
Відкрийте doctype.html у браузері для взаємодії з додатком.
