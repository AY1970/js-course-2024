Проєкт "Бібліотека"
Опис:
Цей проєкт демонструє простий веб-додаток для відображення, додавання, 
редагування та видалення книг на веб-сторінці. Інформація про книги може завантажуватися
з JSON-файлу та динамічно додаватися до HTML-коду за допомогою JavaScript.

Структура проєкту:
/api/books.json: JSON-файл, що містить масив об’єктів, кожен з яких представляє книгу.
/css/styles.css: CSS-стилі для оформлення елементів на сторінці.
/js/async.js: Асинхронний JavaScript код для завантаження даних з JSON-файлу.
/js/controller.js: Контролер для керування рендерингом книг і взаємодії з користувачем 
(додавання, редагування, видалення).
/js/model.js: Дані про книги в форматі JavaScript об’єктів.
/js/view.js: Модуль для відображення списку книг.
/test/booksList.test.js: Тестовий файл для перевірки функціональності рендерингу.
index.html: Головна HTML-сторінка проєкту.
Функціонал:
Відображення списку книг, завантажених з JSON-файлу, на веб-сторінці.
Додавання нових книг через форму на сторінці.
Валідація полів форми перед додаванням книги:
Перевірка на порожні поля.
Перевірка на дублікати назв книг.
Рік видання має бути в межах від 1450 до поточного року.
Редагування існуючих книг.
Видалення книг зі списку.
Динамічне оновлення списку книг після кожної операції (додавання, редагування, видалення).
Оформлення елементів списку за допомогою CSS.
Інструкція з використання:
Клонувати репозиторій на свій локальний комп'ютер.
Встановити всі залежності, запустивши команду: npm install.
Щоб переглянути проєкт у браузері, відкрийте файл index.html.
Щоб запустити тести, використовуйте команду: npm test або npm jest.
