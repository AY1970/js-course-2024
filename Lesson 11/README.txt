# Weather App

Цей проєкт дозволяє отримати поточну погоду в будь-якому місті,
 використовуючи API Open-Meteo та геокодування через Nominatim.

## Структура файлів

- `index.html`: HTML-структура веб-сторінки для введення назви міста та відображення 
   результатів.
- `script.js`: JavaScript для обробки запитів до API та відображення даних на сторінці.
- `README.txt`: Цей файл, що описує функціональність і інструкції.

## Використані API

1. **Open-Meteo**: Погода для координат (широта та довгота). Цей API не потребує API ключа.
2. **Nominatim**: Сервіс для перетворення назви міста в географічні координати.

## Як використовувати

1. Введіть назву міста в поле вводу.
2. Натисніть кнопку "Отримати погоду".
3. Результати відображаються у вигляді назви міста, температури та швидкості вітру.

## Як запустити

1. Завантажте всі файли проєкту.
2. Відкрийте файл `index.html` у своєму браузері.
3. Введіть назву міста у текстове поле та натисніть кнопку "Отримати погоду".

## Вимоги

Для роботи цього проєкту потрібен доступ до Інтернету, оскільки запити до API 
виконуються онлайн.

## Можливі помилки

- Якщо місто не знайдено або введено некоректно, з’явиться повідомлення про помилку.
- У разі проблем з мережею або доступом до API також буде показано відповідне повідомлення.

## Приклад

При введенні міста "Київ", ви отримаєте дані про поточну температуру та швидкість вітру 
у місті Київ.
