README: Проєкт "Бібліотека"

Опис:
------
Цей проєкт демонструє простий веб-додаток для відображення списку книг на веб-сторінці.
Інформація про книги завантажується з JSON-файлу та динамічно додається до HTML-коду за допомогою JavaScript.

Структура проєкту:
------------------
- /api/books.json: JSON-файл, що містить масив об’єктів, кожен з яких представляє книгу.
- /css/styles.css: CSS-стилі для оформлення елементів на сторінці.
- /js/async.js: Асинхронний JavaScript код для завантаження даних з JSON-файлу.
- /js/controller.js: Контролер для рендерингу списку книг на сторінці.
- /js/model.js: Дані про книги в форматі JavaScript об’єктів.
- /js/view.js: Модуль для відображення списку книг.
- /test/booksList.test.js: Тестовий файл для перевірки функціональності рендерингу.
- index.html: Головна HTML-сторінка проєкту.

Інструкція з використання:
--------------------------
1. Клонувати репозиторій на свій локальний комп'ютер.
2. Встановити всі залежності, запустивши команду: npm install
3. Щоб переглянути проєкт у браузері, відкрийте файл `index.html` у вашому браузері.
4. Щоб запустити тести, використовуйте команду: npm jest

Функціонал:
-----------
- Відображення списку книг з JSON-файлу на веб-сторінці.
- Рендеринг книг на сторінці з використанням JavaScript.
- Оформлення елементів списку за допомогою CSS.
